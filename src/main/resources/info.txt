На данный момент бот использует Google Translate API и Yandex.Dictionary API

Тест доступен после того, как ты выучишь минимум 5 слов. Количество вопросов в тесте ограничено диапазоном 1 - 20, и ты всегда можешь досрочно прекратить тестирование, написав команду /exit

GitHub проекта - https://github.com/Exce11ent22/VocabularyBooster_TelegramBot
