Привет! Vocabulary Booster Bot рад тебя видеть! Этот бот даст хороший старт в расширении твоего vocabulary :)

По статистике, 500 самых часто встречающихся английских слов описывают 60% речи. Поэтому зачем учить лишнее?

Этот бот будет выдавать тебе для изучения слова с описаниями на английском языке из особого списка "1000 самых употребляемых слов на английском", что поможет тебе с нуля изучить самые базовые слова языка.

А для проверки твоих знаний бот умеет проводить тесты:
- по последним выученным словам
- по случайным выученным словам
- по словам, которые ты плохо запомнил (основывается на результатах предыдущих тестов)

Но также ты можешь узнать значение любого другого слова, если тебе понадобится, для этого его нужно написать в чат.

Напиши /help, чтобы узнать список команд.

Удачи в пополнении словарного запаса! :)
